# DataBase

🖤 Storage repository for executable files.

Actuellement, ce dépôt contient seulement le projet `AutoBump`.

## Contenu principal
- `AutoBump/src/`
  - `autobump.js`  : script principal Node.js
  - `fix.js`       : script utilitaire
  - `package.json` : configuration npm/build
  - `autobump.exe` : exécutable Windows généré
- `AutoBump/bin/`
  - `autobump.exe` : binaire stocké

## Résumé
Ce dépôt sert à stocker les fichiers et l’exécutable des projet disponible. (`AutoBump`).
